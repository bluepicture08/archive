2. 한국사데이터베이스
개요

국사편찬위원회 한국사데이터베이스는 정보화 시대에 발맞추어 한국사 자료의 디지털화와 웹을 통한 제공을 목적으로 1997년부터 서비스하고 있는 웹 사이트로서, 고대부터 현대까지 한국 역사의 주요 자료를 제공하는 웹 사이트임. 한국사 주요 자료를 이용자가 쉽게 찾고 본문까지 열람할 수 있도록 정제된 데이터를 자료 이용자의 편의에 맞추어 제공하고 있음. 자료의 데이터베이스가 체계적으로 제작되었으며 또한 대부분의 자료가 본문 텍스트까지 구축되어 제공되어 있음. 국사편찬위원회는 자체 예산 확보를 통해 한국사 기초 자료의 데이터베이스화 작업을 꾸준히 진행하고 있음. 2000~2009년 지식정보자원관리사업 및 국가 DB 사업의 지원은 한국사데이터베이스 구축에 큰 도움이 되었음. 모두 98종, 841만 여건, 14억 9천만 여자의 자료(2018년 1월 기준)를 제공하고 있음.

통합검색

- 전체 : 자료 전체에 대해 항목에 관계 없이 자료를 찾음

- 제목 : 제목항목에 대해 검색

- 본문 : 본문영역에 대해 검색

- ID : 한국사데이터베이스 자료는 모두 고유 ID를 가지고 있음. ID를 입 력하여 검색하면 해당 ID의 자료가 검색됨

- 키보드로 입력하기 어려운 한자나 외국어, 기호 등은 문자입력기를 활용하여 원하는 글자를 찾아 입력할 수 있음

상세검색

- 시소러스 검색 : 국사편찬위원회 역사용어 시소러스를 활용한 검색

- 검색연산자의 활용

- 인기검색어의 활용 : 메인 화면의 검색어 입력창 오른편에 표시되는 인기검색어 위에 마우스를 올리면 인기 검색어 상위 10개를 표시함. 이 가운데 하나를 클릭하면 해당 검색어로 바로 검색 가능
