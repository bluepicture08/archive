3.KISS
개요

대한민국 대표 학술지의 서지, 원문 정보를 제공하는 통합검색 시스템으로 현재 3300여 종의 창간호부터 최신호까지 제공함. 전 주제 분야의 서비스를 제공함.

발행기관검색

학회명을 알고 있을 때 유용, KISS발행기관 / 국가지식 발행기관 / 공공저작물

간행물검색

학회에서 발간된 학회지명을 알고 있을 때 유용, KISS 간행물 / 한국연구재단 등재(후보)학술지 / 해외 등재 학술지 / 신규 간행물 / 최신 간행물

주제분류검색

학회지의 주제 분야를 알고 있을 때 유용, KISS분류 / KDC 분류 / KCI 분류

통합검색

제목, 저자명, 간행물명, 발행기관명 통합검색 제공

상세검색

제목, 저자, 간행물, 발행기관 등, 불리언 연산자, ISSN 검색, 주제분야, 발행연도
